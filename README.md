# 📘 Guia de Cursos Online e Gratuitos em TI

## 🚀 Objetivo
Este repositório reúne recursos gratuitos de aprendizado em Tecnologia da Informação (TI).  
O objetivo é orientar estudantes e profissionais sobre caminhos de estudo online, ajudando a desenvolver habilidades técnicas sem custo.

---

## 🧩 Estrutura do Guia

### 1. Fundamentos de TI
- [Introdução à Computação - Fundação Bradesco](https://www.ev.org.br)
- [Lógica de Programação - Curso em Vídeo](https://www.cursoemvideo.com)

### 2. Desenvolvimento de Software
- [Python - Fundação Bradesco](https://www.ev.org.br)
- [JavaScript - FreeCodeCamp](https://www.freecodecamp.org)
- [Java - Udemy Free](https://www.udemy.com/courses/free/)

### 3. Banco de Dados
- [SQL - Khan Academy](https://pt.khanacademy.org/computing/computer-programming/sql)
- [NoSQL - MongoDB University](https://learn.mongodb.com)

### 4. Cloud e DevOps
- [Microsoft Learn - Azure](https://learn.microsoft.com/training/)
- [AWS Training](https://aws.amazon.com/training/)

### 5. Segurança da Informação
- [Introdução à Cibersegurança - Cisco Networking Academy](https://www.netacad.com/courses/cybersecurity)
- [LGPD e Privacidade - ENAP](https://www.enap.gov.br)

### 6. Dados e Inteligência Artificial
- [Machine Learning - Google AI](https://ai.google/education/)
- [Ciência de Dados - DataCamp Free Courses](https://www.datacamp.com)

## 🌐 Plataformas Gratuitas Recomendadas
- [FreeCodeCamp](https://www.freecodecamp.org) – Projetos práticos e certificações gratuitas  
- [Khan Academy](https://pt.khanacademy.org) – Fundamentos de programação e ciência da computação  
- [Microsoft Learn](https://learn.microsoft.com/training/) – Trilhas oficiais da Microsoft  
- [Coursera](https://www.coursera.org) – Cursos gratuitos com opção de certificado pago  
- [Udemy Free](https://www.udemy.com/courses/free/) – Cursos gratuitos em diversas áreas  

---

## 🎯 Como Usar Este Guia
- Escolha uma trilha de acordo com seu objetivo profissional.  
- Dedique tempo semanal para estudo e prática.  
- Combine cursos teóricos com projetos práticos.  
- Documente seu progresso em um portfólio (GitHub, GitLab).
